# ft_transcendence
ft_transcendence is the final group project of the 42 Common Core curriculum. It is a collaborative full-stack web application designed to simulate a real-world production system. The project focuses on teamwork, adaptability, and technical creativity by building a modular platform that includes real-time features, scalable architecture, and modern web technologies.
