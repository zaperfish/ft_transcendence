## Description

**ft_transcendence** is a Meetup-like event platform that enables users to discover, create, and join local events. Built as the final group project of the 42 Common Core curriculum, this full-stack web application simulates a real-world production system with complete architecture and modern development practices.

### Key Features

- **Event Discovery & Management**: Browse upcoming events with detailed information including location, capacity, and timing
- **Event Creation**: Create and organize events with customizable details, capacity limits, and self-defined cover image
- **User Registration**: Sign up and authenticate securely to participate in the platform
- **Event Participation**: Enable joining events and leaving events for event participants and registration management for event creators
- **Real-time Chat**: Communicate with other event participants through WebSocket-based chat rooms
- **Server-Side Rendering**: Optimized page load performance through Next.js SSR
- **Progressive Web App**: Installable mobile experience with offline capabilities and app-like interface
- **Production Monitoring**: Track application health and performance with Prometheus and Grafana dashboards

### Project Goal

This project demonstrates the ability to build a production-ready, scalable web application with proper separation of concerns, containerization, monitoring, and collaborative development practices. It serves as a comprehensive portfolio piece showcasing full-stack development skills, DevOps practices, and team collaboration.
