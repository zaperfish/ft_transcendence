## Features

This section provides a complete list of implemented features with team member attributions and functional descriptions.

| Feature | Description | Contributors |
|---------|-------------|--------------|
| **User Authentication** | JWT-based authentication system with sliding sessions, secure cookie management, and protected routes | yingzhan, xxx |
| **User Registration** | Account creation with validation, password hashing, and automatic authentication flow | yingzhan, xxx |
| **User Profile Modification** | Update user profile information including email address, password and delete user account | xxx |
| **Event Creation (CRUD)** | Create, read, update, and delete events with title, description, location, time, duration, and capacity | yingzhan, xxx |

| **Event Participation** | Enable joining events and leaving events for participants and registration management for event creators | yingzhan, xxx |
